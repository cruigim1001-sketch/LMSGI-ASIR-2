Los tres pilares de la web

Tim Berners-Lee es considerado el padre de la web porque desarrolló los tres elementos básicos para el funcionamiento de la misma: HTML, URI y HTTP.

HTML (HyperText Markup Language, Lenguaje de Marcado de Hipertexto) es el lenguaje de marcado predominante para la creación de páginas web. Fue creado por Tim Berners-Lee en 1991 y ha evolucionado a lo largo del tiempo con varias versiones, siendo HTML5 la más reciente y ampliamente utilizada. HTML utiliza una serie de etiquetas y atributos para estructurar y presentar contenido en la web, permitiendo la inclusión de texto, imágenes, enlaces, formularios y otros elementos multimedia. Es un lenguaje fundamental para el desarrollo web y es interpretado por los navegadores para mostrar las páginas web a los usuarios.
